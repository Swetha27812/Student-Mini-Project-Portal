# Student-Mini-Project-Management-Portal


A modern, full-stack web application designed to streamline the assignment, dispatching, tracking, and evaluation of student mini-projects. Featuring an analytical metrics dashboard, dynamic task dispatch logs, a smooth responsive interface, and an advanced dark mode switching subsystem.

---

##  Features

* **Interactive Dashboard Analytics:** Real-time summary metric cards tracking active deployments, completed assignments, pending reviews, and critical project deadlines.
* ** Dispatch Task Engine:** A tailored, centralized form container designed to deploy new task orders instantly across student teams.
* **Advanced Theme Synchronization:** Persistent, class-based Dark Mode toggles that save your preference seamlessly inside your browser storage (`localStorage`).
* **Frosted Glass UI Aesthetics:** Translucent navigation layouts utilizing Tailwind v4 filters (`backdrop-blur-md`) floating over rich abstract background layouts.
* **Fully Responsive Web Architecture:** Adaptable mobile grid components that scale perfectly from small mobile screens up to wide workstation monitors.

---

##  Tech Stack

**Frontend Framework:**
* React.js (Vite Bundle Engine)
* Tailwind CSS (v4 Reactive Layout Utilities)
* Lucide React (Icon System)
* Axios (Asynchronous HTTP Requests)

**Backend Core Engine:**
* Node.js & Express.js REST API Server
* Cross-Origin Resource Sharing (CORS) Security Layers
